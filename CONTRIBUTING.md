# Contributing to Planta Eco-Farming Context Analyzer

Thank you for your interest in contributing to Planta. This project is open source and welcomes code, documentation, testing, design, and issue-reporting contributions that support the open-source core.

## Scope

Contributions are made to the open-source repository only. Contributions do not grant any rights to proprietary branding, hosted services, commercial API plans, or other non-repository assets unless expressly stated in writing.

## How to Contribute

1. Fork the repository.
2. Create a feature branch.
3. Make your changes.
4. Add tests and documentation where relevant.
5. Submit a pull request.

## Review Process

All pull requests are reviewed by project maintainers. Maintainers may request changes, reject changes, or merge contributions based on project needs, quality, security, and alignment with the roadmap.

## Contribution Standards

Contributions should follow the existing code style, include clear commit messages, and avoid breaking backwards compatibility unless discussed in advance. Security-sensitive changes may require additional review.

## Intellectual Property

By submitting a contribution, you agree that your contribution will be licensed under the MIT License, unless the project later adopts a different contributor policy and clearly announces it.

## Security Issues

Do not open public issues for security vulnerabilities. Report them privately to the maintainers using the designated security contact.

## Code of Conduct

Be respectful, inclusive, and collaborative. Harassment, discrimination, or abusive behavior will not be tolerated.

## Questions

Contact the maintainers through the project issue tracker or the project email address.
